FeedbackLoop Demo

This page demonstrates how to use the FeedbackLoop Chrome extension to send feedback from your website to your AI assistant.

Send Feedback

How It Works

The FeedbackLoop Chrome extension injects a JavaScript API into your web page that you can use to send feedback directly from your code.

Using the API

You can send feedback programmatically using the following code:

// Check if the FeedbackLoop API is available
if (window.FeedbackLoop) {
  // Send feedback
  window.FeedbackLoop.sendFeedback("Your feedback message here");
}

Listening for Responses

You can also listen for responses from the extension:

window.addEventListener('message', function(event) {
  // Only accept messages from the same frame
  if (event.source !== window) return;

  // Check if the message is from our expected format
  if (event.data.type && event.data.type === 'FEEDBACK_LOOP_RESPONSE') {
    console.log('Feedback sent:', event.data.success);
    if (event.data.warning) {
      console.warn('Warning:', event.data.warning);
    }
  }
});
